## Title: Common HTML Tags

There are many HTML tags used, but these are commonly used while creating webpages

```js copy
<html>: Defines the root of an HTML document.
<head>: Contains meta-information about the HTML document.
<title>: Sets the title of the HTML document.
<meta>: Provides metadata about the HTML document.
<link>: Links external resources like stylesheets.
<style>: Defines CSS within an HTML document.
<script>: Embeds or links to client-side scripts.
<body>: Contains the content of the HTML document.
<div>: Defines a division or section in an HTML document.
<span>: Inline container used for styling purposes.
<p>: Represents a paragraph.
<h1>, <h2>, <h3>, <h4>, <h5>, <h6>: Headings of different levels.
<a>: Creates hyperlinks.
<img>: Inserts images.
<ul>: Defines an unordered list.
<ol>: Defines an ordered list.
<li>: List item within <ul> or <ol>.
<table>: Defines a table.
<tr>: Defines a row in a table.
<td>: Defines a cell in a table.
<th>: Defines a header cell in a table.
<thead>, <tbody>, <tfoot>: Table sections for headers, body, and footer respectively.
<form>: Defines an HTML form for user input.
<input>: Defines an input control.
<textarea>: Defines a multiline input control.
<button>: Defines a clickable button.
<label>: Defines a label for an input element.
<select>: Defines a dropdown list.
<option>: Defines an option in a <select> element.
<iframe>: Embeds another HTML document within the current document.
```

## Document Structure

### `<!DOCTYPE html>`

```html copy
<!DOCTYPE html>

<html>
  <!-- Your content goes here -->
</html>
<head>
  <!-- Meta-information about the HTML document goes here -->
</head>
<title>Sets the title of the HTML document</title>
<body>
  <!-- Content of the HTML document goes here -->
</body>
<h1>Heading 1</h1>
<h2>Heading 2</h2>
<!-- ... -->
<h6>Heading 6</h6>
<p>Paragraph.</p>
<p>Line 1.<br />Line 2.</p>
<hr />

### Lists

<ul>
  <li>Unordered list item 1</li>
  <li>Unordered list item 2</li>
</ul>
<!-- Unordered list -->

<ol>
  <li>Ordered list item 1</li>
  <li>Ordered list item 2</li>
</ol>
<!-- Ordered list -->

### Links and Images

<a href="https://example.com">This is a link</a>
<!-- Anchor tag for creating links -->

<img src="https://via.placeholder.com/150" alt="Placeholder Image" />
<!-- Image tag for displaying images -->

### Tables

<table>
  <tr>
    <th>Header 1</th>
    <th>Header 2</th>
  </tr>
  <tr>
    <td>Row 1, Cell 1</td>
    <td>Row 1, Cell 2</td>
  </tr>
  <tr>
    <td>Row 2, Cell 1</td>
    <td>Row 2, Cell 2</td>
  </tr>
</table>
<!-- Table structure -->

### Blockquotes

<blockquote>This is a blockquote.</blockquote>
<!-- Blockquote element for quoted text -->

### Emphasis

<strong>This text is bold.</strong>
<!-- Strong tag for bold text -->

<em>This text is italicized.</em>
<!-- Emphasis tag for italicized text -->
```
